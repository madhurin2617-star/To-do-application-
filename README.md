# 📝 To-Do Application

A simple and efficient command-line based To-Do Application built using Python.  
This project helps users manage daily tasks by allowing them to add, view, complete, and delete tasks easily.

---

# 🚀 Features

✅ Add new tasks  
✅ View all tasks  
✅ Mark tasks as completed  
✅ Delete tasks  
✅ Store tasks using file handling  
✅ Beginner-friendly project structure

---

# 🛠️ Technologies Used

- Python 3
- File Handling
- Lists
- Functions
- Command Line Interface (CLI)
